## 🔬 模型方法论说明

### Dixon-Coles 模型核心公式

```
P(X=x, Y=y) = τ(x, y, λ, μ, ρ) × Poisson(x|λ) × Poisson(y|μ)

其中：
λ = α_i × β_j × HFA（主队i对客队j的期望进球）
μ = α_j × β_i / HFA（客队j对主队i的期望进球）
τ(x, y, λ, μ, ρ) = 1 - λμρ   if x=y=0
                    = 1 - λρ       if x=1, y=0
                    = 1 - μρ       if x=0, y=1
                    = 1 + ρ        if x=y=1
                    = 1             otherwise
```

### 蒙特卡洛模拟设计

2. 每次模拟中引入：
